## 1. 项目愿景

### 1.1 使命

让每一位研究者都能高效地发现、论证与完善自己的研究论题。

### 1.2 愿景

成为学术研究领域的智能助手标杆，通过多 Agent 协作与 AI 技术，为研究者提供从选题到答辩的全流程支持。

### 1.3 核心价值

1. **智能**：多 Agent 协作，模拟真实学术指导
2. **严谨**：五阶段闭环，确保论题质量
3. **开放**：开源开放，社区驱动
4. **可信**：学术诚信优先，防止学术不端

v8.0 (当前)          v9.0 (未来)
